# Install Android File Importer

## Download the App

Get the latest `AndroidFileImporter.zip` from:

https://github.com/icey4826/android-file-importer/releases/latest

Unzip the file, then move `Android File Importer.app` to your `Applications` folder.

## First Launch on macOS

The current app is ad-hoc signed, not Apple-notarized. If macOS blocks the first launch:

1. Right-click `Android File Importer.app`.
2. Choose **Open**.
3. Confirm **Open**.

After the first launch, the app should open normally.

## Connect an Android Device

1. On the Android device, enable **Developer options > USB debugging**.
2. Connect the device by USB.
3. Unlock the device.
4. Approve the USB debugging prompt for this Mac.
5. Use Android File Importer to select files or folders and import them to your Mac.

Android File Importer is read-only with respect to the phone.
